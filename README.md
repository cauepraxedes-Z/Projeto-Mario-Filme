## 📌 Sobre o Projeto
Landing page temática desenvolvida para o lançamento de **Super Mario Bros. O Filme**. Projeto realizado durante os estudos no curso **DevQuest (Dev em Dobro)** para consolidar conhecimentos em desenvolvimento Front-end.

## 🚀 Funcionalidades
- [x] **Vídeo de Fundo:** Fundo dinâmico com loop otimizado para imersão.
- [x] **Modal Interativo:** Sistema de pop-up para assistir ao trailer sem sair da página.
- [x] **Gestão de Áudio:** O vídeo do YouTube encerra automaticamente ao fechar o modal.
- [x] **Design Responsivo:** Layout totalmente adaptado para Desktop, Tablet e Mobile.

## 🛠️ Tecnologias Utilizadas
O projeto foi construído "raiz" (Vanilla), sem frameworks, demonstrando domínio em:
*   **HTML5:** Estruturação semântica.
*   **CSS3:** Flexbox, Variáveis CSS, Media Queries e Animações.
*   **JavaScript:** Manipulação de DOM, Event Listeners e controle de Iframe.
